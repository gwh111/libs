# UITemplateKit

UI组件库，提供UI组件。

提示

定位：和业务无关的，可适用于多种业务的UI工具库。

如充值UI就不算模板库，充值UI要放在对应系统的Kit中，每个通用系统会对应一个Kit，充值对应的业务核心库是AccountLib， 那么相关的UI在AccountKit。 样式和使用指南可参考 iOS UI组件库 和其子目录的文章。

# 主干

pod 'UITemplateKit', :svn => 'https://huored.gicp.net:8088/svn/bench_ios/trunk/UITemplate/UITemplateKit'
